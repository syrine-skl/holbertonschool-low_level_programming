# Function Pointers

This project explores the use of function pointers in C.

## What is a function pointer?

A function pointer is a variable that stores the address of a function.
It allows functions to be passed as arguments to other functions,
enabling dynamic behavior and callbacks.

## Files

| File | Description |
|------|-------------|
| `0-print_name.c` | Prints a name using a function pointer |
| `function_pointers.h` | Header file with all prototypes |
